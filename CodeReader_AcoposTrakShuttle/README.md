# Code Reader ACOPOStrak Shuttle
## Industries:
* Product tracking, needed in all industries


## Task
Define the used 2D-code and read it at the reflective surface

## Solution:
CodeReader


## filname description:

* OK: result OK
* NOK: result NOK
* d: distance object to camera in [mm]
* f: SetFocus
* c: illumination color
* t: exposure time [µs]
* g: gain
